统计学入门课

什么是统计

统计学就是基于数据, 去了解客观现象内在的数量规律性, 从而帮助我们作决策和做预测


为什么要学统计

  • 统计学是高等教育中理工科学生必学的数学
  • 统计学是在几乎所有的领域都发挥这重要作用

例如: 经济学 医学 商业 心理学 计算机


更重要的是培养统计思维:

用相对简单的特征来尽量准确地刻画复杂的真实情况

并且允许对不确定性进行量化

使用统计学的知识和框架了解客观事实的本质和内在规律, 从而帮我们作决策&预测

这四组数据都有一个共性: 它们对事物的性质进行了描述, 所以它们都属于定性数据

数据和变量的关系

数据 (data) 是由变量 (variable) 组成的

下面都是变量, 而他们的取值都是数据

性别 颜色 教育程度 评价
年份 温度 身高 体重

比如说: 在同一个人的身上, 我们可以测量多个变量的数据, 这个人是男是女, 喜欢什么样的颜色, 教育程度如何… …

注意: 一个变量(variable)应该包含至少2个不同的取值, 如果只有一个取值的话则被称为常数


定性 / 分类变量

性别: 男(1), 女(2)

颜色: 赤 (1), 橙 (2), 黄 (3), 绿 (4), 青 (5), 蓝 (6), 紫(7)

我们可以给这些变量标记值


性别和颜色描述的是类别: 是性质上的差异, 而非数量上的差异

虽然我们可以把数字和某一个类别进行对应, 但是男和女的差异不等于一和二的差异

这就是这类变量最本质的特征

由于是对性质类别的描述, 所以只能进行是否相等的比较

无顺序(不能比较大小)或等级(不能比较优劣)

1, 2只是帮我们区分他们是两个类别, 因此我们把这种变量成为分类变量, 更为准确地说是: 无序分类变量



教育程度: 小学(1), 初中(2), 高中(3), 本科(4)…

评价: 好(5), 良好(4), 中(3), 中下(2), 差评(1)


同样, 这两个变量依然是对类别的描述, 是性质上的差异, 而非数量上的差异

但是可以进行是否相等的比较

有顺序(可以比较大小)或等级(可以比较优劣)

但是 好评(5) 不等于5倍的 差评(1)

这是定性变量所不具备的



身高: 1.60, 1.65…

体重: 40, 45, 50 …


这两个变量同样都是数值

可以进行是否相等的比较, 且是有顺序的(可以比较大小)

等间距, 可以进行加减运算

可以进行乘除运算

90 公斤重的人是45公斤重的人的二倍, 以后会讲明是什么决定变量能否进行乘除运算(乘除运算是否是有意义的)


观测, 变量, 与数据矩阵

通常会把数据整理成表格

每一行是一次观测, 一列是一个变量

这样的一个表格我们就称为数据矩阵(data matrix)


测量尺度

为什么要把变量划分地如此细致

测量尺度就好像”尺子”, 不同的尺子可以用来测量不同的事物

直尺可以测量物体的高度, 却不能测量这个物体的重量

体重秤可以测量物体的重量, 却不能测量物体的长宽高

… …

抽象数字系统的属性

不同的”尺子”具有不同的属性, 这些属性使得这些尺子具有去测量某种特定数据的能力

在数学中, 使用抽象数字系统, 这个系统有一些重要的属性, 不同的属性集合就对于了不同的尺子

或者反过来说, 不同的尺子具有不同的属性集合, 因此具有不同的能力


抽象数字系统有4个重要的属性

  1. 每一个数字有其独特的含义 (identity) 例如: 性别 颜色, 我们把1 赋值给男性, 2赋值给女性, 如此, 1 和 2 都有其独特的含义
  2. 数字具有从小到大的固有顺序 (magnitude) 例如: 教育程度 评价

  3. 任意位置的单位间距是相同的 (equal intervals) 例如: 年份 温度

  4. 零(0)表示被测量的属性不存在 (absolute/true zero) 例如: 身高, 体重, 人的身高和体重不可能是0, 如果重量为0就说明重量不存在


测量

  • 测量是指根据一定的规则, 把数字(numbers) 分配给观测 (observations)

比如: 根据一定的规则, 把160这个数字分配给了某一个人的身高

  • 把数字分配给观测的方式, 决定了测量尺度(scale of measurement)

  • 每个测量尺度代表了抽象数字系统的特点属性或属性集合


名目尺度(nominal scale)


名目尺度是根据事物的特征对其进行分类的一种尺度

它只具备抽象数字系统中的第一个属性, 即每一个数字有其独特的含义

如果一个变量能用名目尺度进行测量, 那么这个变量就被称作名目变量 (nominal variable)

例如: 性别, 颜色

之前说这两个变量无序分类变量, 现在又说他们是名目变量, 二者有什么关系呢?

无序分类变量可以理解为这类变量的俗名

名目变量可以理解为学名

俗名告诉我们这个变量可以观察到的一些特性, 而学名告诉我们更加本质的一些东西(每一个数字有其独特的含义)


次序尺度(ordinal scale)

次序尺度是根据事物的特征对其进行等级排序的一种尺度

具备抽象数字系统的前两个属性:

  • 每个数字尤其独特的含义
  • 数字具有从小到大的固有顺序

同理, 如果一个变量是使用次序尺度进行测量, 我们便将其称之为次序变量(ordinal variable)

例如: 教育程度, 评价, 等


等距尺度 (interval scale)

等距尺度是指数值之间的单位间隔是等距的一种尺度

具备抽象数字系统的前三个属性:

  • 每个数字尤其独特的含义
  • 数字具有从小到大的固有顺序
  • 任意位置的单位间距是相同的

如果一个变量可以使用等距尺度测量, 我们则将其称为等距变量(interval variable)

例如: 年份 温度 时间


等比尺度(ratio scale)

等比尺度是具有等距尺度的所有特性, 并且零点有意义的尺度

具备抽象数字系统的所有4个属性:

  • 每个数字有其独特的含义
  • 数字具有从小到大的固有顺序
  • 任意位置的单位间距是相同的
  • 零(0)表示被测量的属性不存在

如果一个变量可以使用等比尺度测量, 我们可以将其称为等比变量(ratio variable)

例如: 身高 体重

有真零点, 因此可以进行乘除运算


如果没有真零点, 则乘除运算结果不唯一(无意义)

例如: 温度(摄氏度) 10, 25 vs 温度(华氏度): 50, 77

对于温度而言, 0是相对的, 在摄氏度下的0和在华氏度下的0并不相等

对同一客观事物从测量会得到不同的值, 而这些值的变化是因为零点的选定不一样而导致的

因此做出的乘除运算不唯一, 也就没有意义去做乘除运算


测量尺度的重要性

测量尺度决定了可以进行哪些逻辑与数学运算和使用哪种统计方法

数值变量还可以进行离散型和连续型的划分

如果一个数值变量的取值是间断的, 两个取值之间在没有其他的值(不可以无限的细分, 就可以将其称为离散型数值变量)

例如: 抛100次硬币, 计算有多少次正面朝上取值只可能是正整数, 不可能是小数. 这就是离散型数值变量


而连续型数值变量的任意两个取值之间都可以进行无限地细分. 比如说身高1.60m 和1.61m

这两个取值之间可以进行无限细分1.601, 1.602…

变量实在哪种尺度上测量得到的, 决定了这些变量能使用哪些逻辑与数学运算, 并且决定了能用哪些统计方法进行分析


更加系统地看待变量

当拿到一个变量时可以进行如下判断

老师，真零值那里没太理解。比如考试的分数从1-100分，这个只能是算等距尺度吧？


如果零点是唯一的，就代表有真零值。比如说身高, 体重, 我们都认同这两个变量的零点是唯一的; 当身高或体重为0的时候, 意味着身高或体重这个客体不存在。零点不唯一的例子是温度, 比如有摄氏0度, 有华氏0度; 当温度为0的时候, 温度这个客体还是存在的。

你问了一个非常好的问题，对于考试分数这个变量来说, 把它当成等比或者等距都可以, 取决于考试打分的规则。如果分数等于回答正确的问题数目占总问题数目的百分比的话(即 正确数目/总数目), 则0值是唯一的, 可以把分数当成等比变量。其实不用过于纠结一个变量到底是等比还是等距, 有一些变量很难界定到底是等比还是等距。我们在分析数据的时候通常把两者都视为数值变量，不加以区分。但是, 当你拿到了数据分析的结果, 要对结果进行解释的时候, 需要考虑一下儿等距/等比这个问题


描述统计


描述统计 (descriptive statistics)

我们可以把统计学分为描述统计和统计推断两个部分

这两个部分正好反应了统计方法发展的前后两个阶段, 也是用统计方法探索客观事物数量规律性的两个不同过程


更进一步, 我们可以把统计推断分为两个问题: 参数估计, 假设检验,

其实统计推断还有一个非常重要的分支: 贝叶斯统计, 贝叶斯统计提供了一种新颖的视角来看待问题和解决问题.


此外概率论也是统计学习中必不可分的一个部分, 它是统计学习的基础


什么是描述统计

描述统计是研究

  1. 如何取得反应客观现象的数据 (数据的收集)

  2. 通过图表形式对数据进行加工处理和可视化

  3. 通过概况与分析得出反应客观现象的规律性数量特征


数据的可靠性和有效性

在分析数据之前, 一定要闻这样的一个问题, 我手中的数据是否可靠(reliable) 和有效(valid)?

  • 可靠性(reliability) : 多次测量得到的数据是否一致, 如果多次测量得到的数据不一致, 那么可靠性是存疑的. 同时可靠的数据不应随着观察者的变化而变化, 所以当多个观察者给出不同回答时, 可靠性也是存疑的
  • 有效性(validity) : 实际测量的对象 = 认为 / 希望测量的对象

点代表我们得到的数据, 靶心代表我们希望测量的客观事物

  • 当数据是可靠并且有效时, 数据会集中在靶心, 可靠说明他们一致性很好(集中), 有效说明测量的是希望测量的客观事物(靶心)
  • 当数据是可靠但无效时, 数据会集中在一起(可靠), 但远离靶心(无效), 它们反应的不是我们希望测量的对象
  • 当数据是有效但不可靠时, 数据会非常分散, 数据的波动性非常大(不可靠) , 但它们又相对集中在靶心周围(相对有效)
  • 当数据既不可靠也不有效时, 数据分散且远离靶心

方差(variance)和标准差(standard deviation)

这是两个度量离散趋势的新指标

方差: 每一个观测值与均值之间的差异的平方和的平均数

而我们通常用 σ 表示标准差, 因此这个式子也表示了方差和标准差之间的关系

方差 = 标准差的平方

因为xi可能小于平均数, 所以减法可能得到负数, 因此我们要对他们进行平方


补充一点:

当我们计算方差时, 分母到底是n 还是n-1

当我们手头的数据是总体, 假如说全世界只有50朵花, 此时50朵花便是总体, 计算它的方差就是除以n(50)

假设有10000朵, 而这50朵是我们随机抽取的, 那么求方差时就应该除以n-1

我们还没有降到概率论中总体和样本的概念, 因此目前我们统一除以n, 在讲述标准差也是除以n

如何处理极端值

  • 如果是由于测量或记录的错误, 或其他明显的原因造成的, 直接丢弃即可
  • 如果极端值出现无法解释, 那么, 丢弃或保留极端值则需要具体问题具体具体分析; 尽量选用受极端值影响较小的指标
  • 可以通过对比保留极端值和丢弃极端值对结果的影响, 来判断结果是否受到极端值的影响


如何处理缺失值

  • 如果含有缺失值的观测记录很少, 而数据量很大, 可以把含有缺失值的观测记录丢弃
  • 如果含有缺失值的观测记录很多, 需要分析原因, 看是否能够把缺失的记录补全
  • 如果含有缺失值的观测记录较少, 可以使用均值/中位数/众数/最大值等进行代替

概率和概率分布

我们已经介绍了描述统计, 之前说过概率论贯穿整个统计学

因此这一章的目的是把一些非常基础但又和统计学密切相关的概率论知识进行介绍


什么是概率论

  • 确定性现象
    • 定义: 在一定条件下必然会发生的现象
    • 抛起的硬币必然下落
    • 在标准大气压下, 水加热到100度必然沸腾
  • 随机现象
    • 抛一次硬币的结果 { 正面, 反面 }
    • 抛一次骰子的结果 { 1, 2, 3, 4, 5, 6 }

随机现象:

  • 在个别实验中结果具有不确定性

  • 在大量重复实验中结果呈现出固有规律性

满足这两个条件的现象就是随机现象

抛硬币 正面 : 反面 ≈ 1 : 1

抛骰子 1 : 2 : 3 : 4 : 5 : 6 ≈ 1 : 1 : 1 : 1 : 1 : 1


概率论和统计就是用来研究随机现象的统计规律性的一门科学

概率论为刻画这种不确定性提供了规律

确定性现象不是概率论研究的对象, 概率论研究的是随机现象


相关术语和概念

  • 样本空间 (Space) (用S来表示)
    • 定义: 随机实验E的所有可能结果组成的集合
    • 例如: 试验: 扔一次硬币进行观察 => S : {Head, Tail}
    • 例如: 实验: 扔两次硬币进行观察 => S : {HH, HT, TH, TT}
  • 随机事件
    • 定义: 样本空间S的子集称为随机试验E的随机事件, 简称事件

例如:

S : {HH, HT, TH, TT} => {HH, HT, TH} = {至少一次正面}

事件发生 : 在本次实验中, 当且仅当这一子集的一个样本点出现

就是说, 扔两次硬币, 看结果是不是上面三个结果之一, 如果是, 则事件发生

样本空间是必然事件

概率

基于个人的经验知识对某件事发生的可能大小进行赋值

首先它可以是纯粹的猜测, 没有任何依据, 仅仅依赖个人的经验/知识, 因此主观色彩非常强, 可信度低

在贝叶斯统计中, 经常使用这种方法, 但在传统的参数估计和假设检验这两个问题下不太会用到这种方法


频率

例如

性质:

  • 0 <= fn(A) <= 1

  • fn(S) = 1 (样本空间是必然事件)


概率

对于随机试验E的每一个事件A赋予一个实数, 记作P(A), 称为事件A的概率, 如果集合函数P(.)满足下列条件

  • 非负性: 对于每一个事件A, 有P(A) >= 0
  • 规范性: 对于必然事件S, 有P(S) = 1
  • 可列可加性: 设A1, A2… 是两两不相容事件, 有P(A1 U A2 U …) = P(A1) + P(A2) + …


频率 vs 概率

为什么频率近似概率

随着n的增加, 频率逐渐稳定于某个常数, 这就是统计规律性, 这个常数就是概率

当实验次数非常小时, 频率波动非常大

当实验次数非常大时, 频率波动非常小, 趋近于常数

小数定律: 当重复实验的次数非常小时, 频率不能非常好地代表事件发生的可能性大小, 我们不能使用它很好地近似概率

大数定律: 当实验次数n趋于无穷时, 事件A发生的频率趋于事件A发生的概率

正态分布的特征

并不是所有这样的曲线都叫正态分布, 正态分布有着独有的特征

  • u - σ, u + σ 之间的面积为 68.26%, 也就是说, 在正态分布中, 数值落在(u - σ, u + σ) 的概率为 68.26%

  • u - 2σ, u + 2σ 之间的面积为 95.44%, 也就是说, 在正态分布中, 数值落在(u - 2σ, u + 2σ) 的概率为 95.44%

  • u - 3σ, u + 3σ 之间的面积为 99.74%, 也就是说, 在正态分布中, 数值落在(u - 3σ, u + 3σ) 的概率为 99.74%

因此正态分布有3σ法则

3σ法则: 正态变量的取值范围是正负无穷, 但它的值几乎肯定落在(u-3σ, u+3σ)

满足这些性质的, 同时长成这个样子的曲线, 才叫正态分布

样本和抽样分布

统计推断问题可以化为参数估计问题和假设检验问题

为什么要做参数估计和假设检验?


总体与样本

总体(Population) : 我们所感兴趣的所有对象的集合

在有限的时间内, 有限的预算下我们通常无法获得所有对象的信息

因此一个自然的想法就是: 拿到其中的某些数据, 基于这些数据对总体进行推测 (样本 sample)

基于同一个总体我们可以获得很多样本, 每一个样本都能提供一些关于总体的信息

参数估计, 估计的就是总体的一些参数, 例如总体的平均收入

不同的样本提供的信息是有一定差别的

如何使用样本的信息去得到总体的参数的信息?

解决这个问题的过程就叫做参数估计

而当得到参数估计以后, 验证这个估计对不对, 或者有多大的可能性是正确的, 就是假设检验


总体(Population)

总体: 试验的全部可能的观察值

个体: 每一个观察值

总体的容量: 总体中所包含的个体的个数

由于总体中每一个个体都对应一个随机试验的观察值, 那么一个总体就对应于一个随机变量X, 每一个个体的取值就是随机变量X的取值

因此我们可以不用刻意区分总体和随机变量, 而是统称为总体X

对总体的研究就是对一个随机变量X的研究, X的分布函数和数字特征就成为总体的分布函数和数字特征


样本(Sample)

在实际中, 总体的分布一般是未知的, 或只知道它具有某种形式而其中包含未知参数

从总体中抽取一部分个体, 根据获得的数据来对总体分布做出推断. 被抽出的部分个体叫做总体的一个样本

样本是进行统计推断的依据


以其中的一个样本为例讲解:

从总体抽取一个个体, 就是对总体X进行一次观察并记录其结果 (因为总体是由个体组成的, 如果我们直到所有个体的情况, 那么我们自然也就了解了总体)

在相同的条件下对总体X进行n次重复的, 独立的观察, 将n次观察结果按试验的次序记为 X1, X2 … Xn

因为X1…Xn是对随机变量X的观察结果, 并且各次观察实在相同的条件下独立地进行的, 因此我们可以认为X1, X2…Xn是互相独立的

互相独立: X1 的取值并不能告诉我们任何 X2 的取值的信息

X1…Xn是与总体X具有相同分布的随机变量

就是说Xi本身就是一个随机变量, 它可能的取值和取值出现的概率都和总体X是一致的

Xi代表总体中某一个个体, 在我们真正在这个个体上得到观察值之前, 它可能的取值就和总体可能的取值是一致的, 并且它所对应的观察值可能出现的概率也和总体是一致的

因此我们任务X1…Xn每一个都是随机变量, 并且与总体具有相同的分布

当n次观察完成我们就得到了一组实数x1, x2…他们是X1, X2…Xn的观察值, 称为样本值



抽样分布

我们需要使用样本对总体的参数进行估测, 但其实我们往往不是直接使用样本本身, 而是针对不同的问题构造样本的适当函数, 利用这些样本的函数进行统计推断

而这个样本的函数有个专有的名词统计量


统计量

定义:

左边就是一个样本, 右边是一个统计量, 从样本到统计量我们需要一个函数(g)

这里g的参数是一组变量

一旦我们对X1, X2…进行观测并记录观测值, 我们便会得到一组实数x1, x2…(样本值)

如果我们把x1, x2…作用到g, 由于此时g没有未知参数, 我们便可以得到一个函数值(统计量的观察值)


常用统计量

由于是对样本进行操作, 所以分母为n-1, 而不是n

具体在参数估计章节讲解


小结

基于总体, 我们可以得到许多样本, 对每一个样本我们都可以得到一个统计量, 以样本均值为例, 我们可以有很多样本均值的观察值

这些统计量的观察值也会形成一个分布, 称为统计量的分布, 即抽样分布



常用统计量的分布

来自正态总体的几个常用统计量的分布

总体服从正态分布, 基于总体我们可以得到样本, 基于样本我们可以构造样本的函数(统计量), 基于统计量我们又可以得到统计量的分布

这里我们将要讲解三个常用的统计量的分布

  • 卡方分布
  • t分布
  • F分布


卡方分布(chi-square distribution)

首先我们有一个总体, 它是标准正态分布. 基于这个总体我们可以抽取一部分个体X1, X2…Xn, 基于这个样本我们就可以构造统计量

这个统计量的计算方法就是对每一个个体进行平方再加和

这样构造出来的统计量就服从自由度为n的卡方分布

自由度(df: degree of freedom) = 独立变量的个数


卡方分布的概率密度函数图

可以看出来, 随着自由度的增加, 卡方分布的概率密度函数图变得越来越矮, 也越来越胖

那么我们也就看出来了, 同样是基于样本构造出来的卡方统计量, 它的分布到底长什么样子也取决于样本容量

所以样本容量, 也就是自由度, 是一个非常重要的指标


卡方分布的性质

对于卡方分布, 它的均值就等于n, 方差等于2n (不是重点, 记住即可)


t分布(t distribution)

同样是一个服从标准正态分布的总体, 从这个总体中可以得到两个变量 X 服从标准正态分布,Y服从自由度为n的卡方分布


在此基础上基于X,Y就可以构造一个新的统计量 t


t分布的概率密度函数图

黑线是标准正态分布的图, 有颜色的线对应的是t分布的图, 把二者画在同一图中是为了方便比较

随着自由度的增大, t分布越来越接近标准正态分布, 对称轴为x=0


F分布(F distribution)

同样是一个服从标准正态分布的总体, 通过这个总体可以得到一些统计量, U服从自由度为n1的卡方分布, V服从自由度为n2的卡方分布


基于U和V构造新的统计量F


F分布的概率密度函数图

首先固定n2为40, 去变换n1

随着n1的不断变小, 图形变得越来越矮, 越来越胖, 并且其概率最大值所对应的F值也越来越向左偏移


固定n1为20, 变化n2, 一次递减10

随着n2递减, 图形变得越来越矮, 相对的也胖了一些, 但变换程度没有左侧的明显



正态总体的样本均值和样本方差的分布

样本均值的期望, 或者说样本均值这个统计量的均值 等于u,

而样本均值这个统计量的方差就等于 σ^2 / n, n为样本容量

样本方差S^2, 这个统计量的均值等于σ^2


这几个等式告诉了我们什么?

X bar和S^2都是基于样本计算出来的统计量

而 u 和 σ^2 却是总体的参数, 分别为总体的均值和总体的方差

我们之前说过几乎不可能拿到总体所包含的所有个体数据, 所以我们也基本上去真实地计算出来总体的均值和总体的方差

如果我们想要了解这个总体的均值和方差的话, 我们使用的办法就是对总体进行观测, 具体的步骤就是从总体中抽取一些个体, 对这些个体进行观测, 并从所得到的数据中提取一些信息. 基于这些信息我们来对总体的参数(例如: 均值, 方差)进行估计

而这个几个等式, 其实就已经把总体的参数和样本的均值, 以及样本的方差之间建立了联系, 使我们能够基于样本的信息推测出总体的参数

注意: 我们没有限定总体X的分布, 没有说必须是正态总体

也就是说, 不管总体服从什么样的分布只要样本均值为u, 它的方差为σ^2, 那么我们都可以用样本的均值和样本的方差对总体的均值和总体的方差进行估计

如何证明这些等式是正确的?

这里简单证明一下样本均值的期望等于u, 以及样本均值的方差等于σ^2/n


定理

样本均值这个统计量的分布为什么服从正态分布

中心极限定理(Central Limit Theory) 告诉我们样本均值的分布一定服从正态分布

中心极限定理(CLT)

中心极限定理是概率论中的一组定理

中心极限定理说明, 在适当的条件下, 相互独立的随机变量之和经适当标准化后, 其分布近似于正态分布; 注意: 不要求变量本事服从正态分布


样本均值的分布

样本来自正态总体

红色的线x=0是理论上的总体均值, 因为我们的总体服从标准正态分布, 所以它


随机抽样, 误差源, 随机分配

样本是由从总体中随机抽取的个体组成的, 随着随机抽取的个体的不一样, 我们就可以得到不同的样本

虽然样本可能是有变化的, 但是好的样本的标准却是一致的: 我们希望样本是具有代表性的

我们希望基于样本得到的结果是可以很好地代表总体上的结果的


误差源

在这个过程中有一些误差源, 这些误差源会影响样本的质量

  • 方便性: 容易联系到的人更可能包含进来
  • 不回复: 随机抽取的样本中只有一部分回答了问卷
  • 自愿性: 样本由自愿参与的人组成


区分观察研究(Observational Study)和实验(Experiment)



参数估计

点估计

设总体X的分布函数的形式已知, 但它的一个或多个参数未知, 借助于总体X的一个样本来估计总体未知参数的值的问题称为参数的点估计问题

从总体中获取样本, 基于这个样本构造一个统计量, 基于这个统计量得到的观察值就认为近似于总体的未知参数


估计量, 估计值


对于同一参数, 使用不同的估计方法求出的估计量可能不相同. 采用哪一个估计量好?

  • 无偏性: 若估计量的数学期望存在, 并且该期望等于总体参数, 则称为无偏估计
  • 均值vs期望: 均值是一个统计量(基于样本构造的函数); 期望完全由随机变量的概率分布所确定(“上帝视角”); 两者常混用
  • 对于某些样本值, 由这一估计量得到的估计值相对于真值来说偏大或偏小, 但是反复将这一估计量使用多次, 就”平均”来说其偏差为0
  • E(估计值) - 真值 称为系统误差; 无偏估计的实际意义就是无系统误差
